Nosso projeto é uma cartilha de vacinação infantil (0-19) onde o médico cadastra o paciente e cria sua tabela de vacinas ou a atualiza, com funções cadastro/alterar/visualizar.

Temos no cabeçalho dois artigos informativos sobre vacinas e o logo do grupo ImmunoSync que é um botão clicável que volta para tela inicial de cadastro.

a pagina alterar cartilha altera os dados cadastrados

a pagina cadastro cria um novo usuario

e a pagina cartilha apenas visualiza a cartilha pesquisada por id 

as páginas cadastrar, alterar cartilha e  visualizar cartilha tem botões que após a consulta pode redireciona-los para qualquer dessas páginas e o cabecalho com o home.
